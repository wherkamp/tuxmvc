## Installation
#### Maven
```xml
<repository>
   <id>kingtux-repo</id>
   <url>http://repo.kingtux.me/repository/maven-public/</url>
</repository>

<dependency>
   <groupId>me.kingtux</groupId>
   <artifactId>tuxmvc-core</artifactId>
   <version>1.0</version>
   <scope>compile</scope>
</dependency>
```
#### Gradle
```java
repositories {
  maven { url 'http://repo.kingtux.me/repository/maven-public/' }
}

dependencies {
   compile "me.kingtux:tuxmvc-core:1.0"
}
```
